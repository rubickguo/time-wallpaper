# 时间壁纸

把沉在硬盘里的旧照片，变成每天都能重新遇见的私人记忆壁纸。

时间壁纸是一款面向 Windows 的本地照片应用。它会从你的照片库里挑出适合“那年今日”的十个瞬间，为每张照片生成克制、文艺、不过度编造的短文案，并支持把这些照片作为桌面壁纸循环展示。

它适合那些照片很多、但很少真正回看的人：应用负责筛选、排版和循环展示，照片仍留在本地，文案只在你明确配置 LLM 后生成。

English version: [README.en.md](README.en.md)

## 项目亮点

- **像一封旧日来信**：每天只呈现十张照片，降低回看负担。
- **优先保护真实记忆**：不编造人物关系、地点和情绪，不用油腻或冒犯性的表达。
- **面向桌面壁纸优化**：按物理分辨率生成成品图，尽量避开常见图标区域。
- **兼容多种模型服务**：支持 OpenAI-compatible 和 Anthropic-compatible 接口。

## 核心功能

- 选择一个或多个本地照片文件夹，递归扫描 `jpg/jpeg/png/webp/bmp/gif` 图片。
- 优先挑选日期接近“那年今日”、分辨率高、清晰、适合做壁纸的照片。
- 当精确日期照片分辨率不足时，会自动扩大到前后几天寻找更适合 4K 壁纸的候选。
- 支持 OpenAI-compatible 和 Anthropic-compatible 接口，可接入百炼、MiMo、中转商、OpenRouter、One API 等服务。
- 使用稳定的 Prompt 管理和输出校验，减少事实编造、油腻文案、隐私冒犯和重复句式。
- “旧日来信”只展示十张精选照片，每张照片带一条 AI 文案。
- 支持桌面壁纸循环，每小时切换一张，共十张。
- 生成壁纸时会把文案和日期绘制进最终壁纸，默认放在右下角，尽量避开常见桌面图标区域。
- 适配主屏真实物理分辨率，支持 1080p、2K、4K、多屏和横竖图混排。

## 使用方式

### 开发运行

```powershell
npm install
npm run dev
```

### 构建 Windows 版本

```powershell
npm run dist
```

构建产物会输出到 `release/` 目录，包含安装版和 zip 包。

## LLM 配置

在应用内打开 `LLM 设置`：

- 协议：
  - `OpenAI-compatible`：适用于 OpenAI、阿里云百炼兼容模式、MiMo OpenAI-compatible、OpenRouter、One API 等。
  - `Anthropic-compatible`：适用于 Anthropic Messages 兼容接口。
- Base URL：填写服务商提供的兼容接口地址。
- API Key：填写对应服务的密钥。
- 模型 ID：填写服务商实际支持的模型名。
- 视觉模式：
  - 推荐使用支持视觉的模型，让模型读取图片并生成更贴合画面的文案与评分。

## 文案生成链路

```text
图片 -> 本地 EXIF/质量分析 -> 结构化图片信息 -> LLM 生成文案和评分 -> 程序校验 -> 缓存 -> 展示/壁纸生成
```

文案生成强调：

- 只基于图片和结构化信息，不编造人物关系、地点、事件或情绪。
- 不使用“家人、恋人、朋友、旅行、幸福、遗憾、告别、永远”等容易冒犯或过度推断的表达。
- 每张照片生成多条候选，但首页和壁纸默认使用第一条主文案。
- 十张图片之间尽量避免相似句式。

## 隐私说明

照片默认保存在本地。只有在用户配置并启用 LLM 后，应用才会按配置向服务商发送必要的图片或结构化信息。请根据所接入服务商的隐私政策自行判断是否开启视觉模式。

## 发布

推荐发布流程：

1. 更新版本号。
2. 运行 `npm run dist`。
3. 将 `release/` 中的 Windows 安装包和 zip 包上传到 GitHub Release。

如果本机已经通过 `gh auth login` 登录 GitHub，也可以直接运行：

```powershell
npm run release:github
```

该命令会构建 Windows 包、提交当前代码、推送 `main` 和版本 tag，并创建或更新 GitHub Release。
